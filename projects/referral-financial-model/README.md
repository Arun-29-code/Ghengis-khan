# Referral Programme Financial Model

Interactive financial model for the GP Automate practice referral programme. Self-contained single-file React app — open `index.html` in a browser and it works. No build step, no install required. All dependencies (React, Tailwind, Recharts, prop-types) load from CDNs at runtime.

## Contents

- 9 tabs covering executive summary, scenario comparison, cashflow, 3-year model, tier breakdown, volume discount, channel comparison, assumptions & audit, and a board/investor narrative view
- Reward formula based on referee contract value
- Four-band practice size distribution
- Full P&L cost model

## History

Originally built on the orphan branch `claude/nhs-referral-financial-model-VJ78l` in a previous Claude Code session. Relocated into `main` on 2026-04-09 so it lives inside the standard repo structure.

## How to open

Double-click `index.html` or run `open index.html` from the terminal in this folder. It will open in your default browser. Internet connection required (for the CDN scripts).
